### 👋🏻 Welcome to Citrolabs

Citrolabs builds Ego, a browser-based personal assistant for working with AI agents across the web. Ego helps users understand context, complete tasks, and collaborate directly inside browser workflows. Ego Lite is the community edition for individual users and builders.
